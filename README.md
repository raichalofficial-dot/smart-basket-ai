# 🛒 Smart Basket AI

Smart Basket AI is an intelligent supermarket management and shopping assistance system developed using Django and AI-based features. The project helps customers, sellers, delivery partners, and admins manage shopping activities efficiently with smart recommendations and analytics.

## 🚀 Features

### 👤 Customer Module

* User registration and login
* Product browsing and smart search
* AI-based budget cart suggestions
* Pantry tracking system
* Price comparison
* Seasonal pack recommendations
* Cart and wishlist management
* Order tracking and notifications
* Family group shopping

### 🏪 Seller Module

* Product and category management
* Smart restock assistant
* Product performance analysis
* Complaint handling
* Seller dashboard

### 🚚 Delivery Module

* Delivery partner profile management
* Live order tracking
* Delivery assignment system
* ETA prediction using AI

### 🛠️ Admin Module

* Manage users and sellers
* Delivery monitoring dashboard
* Complaint management
* Sales and seller reports
* Commission tracking

## 🤖 AI Features

* Smart product search
* AI pantry tracker
* AI budget cart planner
* Complaint priority prediction
* Delivery ETA prediction
* Product recommendation system

## ▶️ How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/raichalofficial-dot/smart-basket-ai.git
```

2. Move to project directory

```bash
cd smart-basket-ai
```

3. Install requirements

```bash
pip install -r requirements.txt
```

4. Run migrations

```bash
python manage.py migrate
```

5. Start the server

```bash
python manage.py runserver
```

6. Open browser

```bash
http://127.0.0.1:8000/
```
